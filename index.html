📋 INSTALLER L'AGENT PC
1. Télécharge le script
wget [URL du script depuis Vercel]/sutur_agent.py
2. Installe les dépendances
pip install websockets pillow pyautogui
sudo apt install scrot xdotool -y
3. Lance l'agent
python3 sutur_agent.py --token TON_TOKEN
ℹ️ Une fois lancé, ouvre CONFIG → REMOTE DESKTOP → Ouvrir. Stream démarre automatiquement.
